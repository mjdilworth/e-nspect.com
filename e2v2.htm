Real Time Reporting Software for Exchange 2000

With e-nspect2000 you can now determine just how your messaging systems are being used. e-nspect2000 measures and reports on all internal and Internet mail. e-nspect2000 provides managers with much needed information on which to base commercial decisions and lets system professionals know how their systems are scaling.

  • Real Time data collection and reporting.
  • Over 20 key reports of messaging use (including time series, current activity, host, domain, user, attachments).
  • Easy to install and use.
  • Scales from a single server to an Enterprise wide multi server environment.
  • Uses Microsoft Management Console (MMC) technology, thereby minimising any learning curve.
  • The e-nspect2000 licensing structure makes for an affordable solution.
