FreightMetrics Toolbar

The FreightMetrics Excel Toolbar is used just as any other Excel toolbar. The toolbar can be added or removed from Excel using the View->Toolbars menu. The toolbar can then be manipulated just like any other Excel toolbar, it can be docked or it can float.

The FreightMetrics toolbar provides three main functionalities, which are:

  • To open the excel function wizard that can be used for Fr8Tools functions.
  • To open Fr8Tool Excel templates.
  • To open the Fr8Tools Data Manager.
