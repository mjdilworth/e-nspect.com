Forward Curve Builder - Input & Output Parameters

Below we briefly describe the input and output parameters of the Forward Curve Builder*:-

Contract: The program allows the user to choose a contract from the drop down menu. There are certain logical rules that should be applied when building your forward curve. For instance;

. The contracts chosen should be sequential and there should be no overlapping. For example, Q4-05 should come before Q1-06 in the list.

. The user will obtain better 'price discovery' if he uses all the FFA prices available to him. For instance by obtaining and inputing prices for Q1-06 and Q2-06 contracts seperately instead of Q1&Q2-06 or by inputing prices for the Q1&Q2-06 and Q3&Q4-06 contracts instead of CAL-06 will lead to a more accurate/realistic 'forward curve'.

Period (From) & Period (To): As an aid to the user the system will automatically fill in the appropriate dates for the contract

Price: The user should input the market price of the contract manually into each appropriate box.

Type of Curve: This input box allows the user to choose between a 'Term Structure' (in simple terms the system calculates every price on a daily basis reflecting the implied spot rate for the term of the market chosen. For a more detailed explanation of 'Term Structure' please refer to the FreightMetrics document 'Freight Forward Curves: Techniques and Applications). In reference to the 'Forward (implied spot)'calculation - here the system calculates each daily price reflecting the implied future spot rate for a particular date.

Interpolation Method: FreightMetrics' systems use the cubic-spline formula to interpolate as preferred methodology.

Frequency: The user can calculate on a daily, weekly or monthly basis by inputing 1 for daily, 7 for weekly etc

Spot Price: Here the user should input the spot price of the market under observation.

Single Date Calculation: Here the software allows the user to calculate the forward rate for a single specific date.

* Note: For users of the FreightMetrics MonteCarlo (Fr8) model for option pricing - it is essential that the user creates a forward curve on the 'Forward Curve Builder' template first and then transfers (simply by using a cut & paste operation) the data into the FreightMetrics Option Calculator.

Implied Forward Rates - Input & Output Parameters

Below we briefly describe the Implied Forward Rates template.
Implied Forward Rates: As the title implies this template allows the user to calculate implied forward rates of a contract for specific monthly and quarterly dates.

Copy from Builder: This function allows the user to automatically extract data from the 'Forward Curve Builder' and place it in the 'Term Structure' columns of the 'Implied Forward Rates' template.

Note: The user must have the 'Term Structure' box appear next to 'Type of Curve' in the Calculation Parameters section of the Forward Curve Builder template before using the 'Copy from Builder' function.

Calculation Parameters: Here the user should input a specific 'beginning date' and 'ending date' of the forward curve being studied. The 'Implied fwd rate' output in cell E10 is the average implied forward rate for the period specified by the user.
