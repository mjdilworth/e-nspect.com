xlGreeks_Asian_MC

This function calculates the risk statistics (delta, gamma, vega...) for Asian style options on freight using a generic Monte Carlo simulation model. For further information on how the corresponding option premium is obtained for the purpose of estimating the risk sensitivities, please refer to the function xlAsian_MC.
Argument Description Sample Data Switch
keepRandom Keep previous random numbers FALSE FALSE, TRUE
result Selection of required risk statistic 2 2=Delta, 3=Gamma, 4=Vega, 5=Theta, 6=Rho
valueDate Value date 01/09/2005
avgBeg Start of average period 01/01/2006
avgEnd End of average period 31/03/2006
optType Type of option (call/put) 1 1=Call, 2=Put
strike Strike price 22000
FFAprice Price of the underlying FFA contract 20000
vol Volatility (annualised) 40% or 0.40
intRate Risk-free interest rate 4% or 0.04
simRuns Number of simulation runs 5000

References
  • Glasserman, P. (2004), Monte Carlo Methods in Financial Engineering, Springer.
  • Jaeckel, P. (2003), Monte Carlo Methods in Finance, Wiley.
  • Tavella, D. (2002), Quantitative Methods in Derivatives Pricing, Wiley.
