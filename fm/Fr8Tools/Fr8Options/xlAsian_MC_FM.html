xlAsian_MC_FM

This function calculates the fair value for Asian style options on freight using an extended Monte Carlo simulation model developed by FreightMetrics to account for the full forward curve information. The main difference of this function compared to the xlAsian_MC function is the fact that xlAsian_MC_FM generates random price paths around the full (daily) forward curve of the underlying asset instead of simply taking the corresponding FFA price (or expected average spot price) for the settlement period of the option. In other words, it prices a freight option based not only on the expected average level of future spot prices, but also on the expected path of the underlying spot price (as implied by the shape of the full forward curve). (For further evaluation of the various option pricing models offered by Fr8Tools, please refer to the document “Option Pricing Model Comparison”)
Argument Description Sample Data Switch
keepRandom Keep previous random numbers FALSE TRUE, FALSE
valueDate Value date 01/09/2005
avgBeg Start of average period 01/01/2006
avgEnd End of average period 31/03/2006
optType Type of option (call/put) 1 1=Call, 2=Put
strike Strike price 22000
fwdCurve Forward curve of the underlying asset see note below
vol Volatility (annualised) 40% or 0.40
intRate Risk-free interest rate 4% or 0.04
simRuns Number of simulation runs 5000

Note: The fwdCurve argument is passed on in the form of an Excel range consisting of 2 columns. The first column contains dates and the second column contains the corresponding forward prices. No headings are required at the top row of the range. The range can be generated from the xlFwdCurve_full function (with curveType=1).
