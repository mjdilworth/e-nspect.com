License

Fr8Tools is licensed on a per host (computer) basis. Therefore, a unique license is required for every machine that Fr8Tools is installed on.
By default Fr8Tools is installed as a time limited version. Upon completing the installation the software will function for 7 days, after which it will disable itself.

The software can only be re-enabled with the application of a suitable license key obtained from FreightMetrics.

Before you can receive your license key, a Host ID has to be produced using the Fr8Tools toolbar.

From the Fr8Tools Excel menu bar, under the Help option, the license data interface (as illustrated) can be used to generate both your Host ID and to enter the license code returned by FreightMetrics.
