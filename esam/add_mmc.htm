e-nspect2000 Event Sink Administration Module - Adding to MMC

The e-nspect Event Sink Administration Module is easily added to the Microsoft Management Console (MMC).
From the console menu of the MMC select the "Add/Remove Snap-in" option. This will display a new window with two tabs. Using the "Standalone" tab select the "Add" button to display the "Add Standalone Snap-in" dialog. From here select the "e-nspect Event Sink Administration" snap-in, as illustrated below.

PLEASE NOTE:

Incorrect use of the e-nspect Event Sink Administration Module can seriously damage you SMTP installation.
