e-nspect2000 Event Sink Administration Module

The e-nspect Event Sink Administration Module has been designed to provide you with a familiar and easy to use interface with which to manage Simple Mail Transfer Protocol (SMTP) event sinks.

The e-nspect Event Sink Administration Module is a Microsoft Management Console (MMC) snap-in and can be used to manage the event sinks on both the Windows 2000 server SMTP server and Exchange 2000 server.

Using the e-nspect Event Sink Administration Module you can:

  • View all the installed event sinks.
  • Add new event sinks.
  • Delete event sinks.
  • Edit event sink properties.
  • Delete SMTP instances.
  • Backup and Restore the SMTP event sink configuration data (Metabase).

PLEASE NOTE:

Incorrect use of the e-nspect Event Sink Administration Module can seriously damage you SMTP installation.
