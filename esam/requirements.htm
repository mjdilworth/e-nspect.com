e-nspect2000 Event Sink Administration Module - Requirements

The e-nspect2000 Event Sink Administration program has the following requirements.
  • The snap-in has been tested on the following platforms:
    • Windows NT4 Server.
    • Windows 2000 Server.
    • but it should run on most Windows platforms.
  • The snap-in requires version 1.2 of the Microsoft Management Console.
  • The snap-in can manage the following SMTP servers:
    • Microsoft Exchange 2000.
    • Windows 2000 Server.
PLEASE NOTE:

Incorrect use of the e-nspect Event Sink Administration Module can seriously damage you SMTP installation.
