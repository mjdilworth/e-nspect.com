Flushing the cache

All SQLCache cache can be flushed from the MMC, or if appropriate, cache at the service and specific command levels can be individually purged. What is more, these operations can be performed individually on a per host basis, or for all hosts connected to in the MMC.

By selecting the "All Tasks" menu option on the main SQLCache Administration node you can choose to purge SQLCache's cache at the varying configuration levels. Flushing the cache from the SQLCache Administration node purges the cache from all the hosts currently showing(connected) in the MMC console.
