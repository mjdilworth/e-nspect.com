Adding and Removing Accelerated Host to the SQLCache console

After the SQLCache snap-in has been added to the MMC, an accelerated host (a machine on which SQLCache has been installed) can be added to the console, from where it can be configured and managed.

Selecting this option displays a dialog box from where the host name (or IP address) is entered. Alternatively, you can browse the network for the computer.

The added host can then be administered and configured using the SQLCache snap-in.

A host can be removed from the MMC console by selecting the "Remove host" option, of the "All Tasks" menu, from the host node. this is shown below.
