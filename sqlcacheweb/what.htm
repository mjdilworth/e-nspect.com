What is SQLCache?

SQLCache is an OLE DB data provider with an in-built and fully tunable data cache.

This means that you can use it like any other ADO data provider in your code. SQLCache does not require your programmers to learn any new techniques or Application Programming Interface (API). Importantly, existing code can use SQLCache simply by changing the ADO connection string.
