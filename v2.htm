"You too can easily have reports like these"

  • Over 20 key reports of messaging use (including time series, current activity, host, domain, user, attachments).
  • Easy to install and use.
  • Scales from a single server to an Enterprise wide multi server environment.
You want to know your current usage scenario, but you also need to know if its "normal". Use the time series reports to identify trends, peaks and potential problems. Trend analysis is available for the current day, month, year and year history for messaging and attachment usage.

You want to know what messaged have been flowing thhrough your Exchange server in the last hour. Who sent them to who, what was the subject, the message size, the attachments. You can use the Current Activity report to see details like these.

You want to know what are the top domains your server communicating with. Select the Domain report and then click the messge Quantity column to sort. All columns can be used to sort in this manner.

Select a domain to "drill down" and get even more data.

Find your top users in the same way. Select the User Reports, then sort on any column, simply by selecting it.

But.. what are these top users really doing? To find out simply expand the user to get the user's details for Sent, Received, Domain, IPM, and attachments.

Many more reports are available..this was just a quick peek!
