Query - Current Activity Reports

The Current Activity Report details all the messages that have passed through your monitored systems over a certain time period. The default time period is the last 24 hours, however this is a completely configurable feature. Furthermore, the reports can be viewed by SMTP host and per virtual SMTP instance.
From the Current Activity report node you can set the date range for the report generation, print the report, or save the report to a HTML file.
