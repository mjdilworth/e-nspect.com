Query - User Reports

By selecting and expanding the User Reports node you can view summary and full user reports. All columns can be sorted bi-directionally.

From the User report node you can set the date range for the report generation, print the report, and save the report to a HTML file.

Summary User Report.

Full report per user contains the following information for both sent and received messages:
  • Date of message.
  • Subject of message.
  • Recipient Address(es).
  • Recipient Display Name(es).
  • Message Size(bytes).
  • Number of Attachments.
  • Attachment Name(s).
  • Attachment Size (bytes).
Also, user data is further broken down into seperate domain, IPM and attachment reports as seen in the host reports.
