e-nspect2000 Query Tour

e-nspect2000 Query is the graphical tool used to view the collected data from Exchange 2000 server SMTP instances.

The Query tool is a Microsoft Management Console (MMC) snap-in and can connect to multiple e-nspect database servers.

Using the Query tool you can see the following reports:

  • Time Series (Daily, monthly, yearly) reports for all Data.
  • Time Series (Daily, monthly, yearly) reports for all Data by SMTP host.
  • Time Series (Daily, monthly, yearly) reports for all Data by Virtual SMTP instance.
  • Summary reports for all SMTP hosts.
  • Summary reports for all virtual SMTP instances.
  • Summary message report for each SMTP server.
  • Domain report for each SMTP server.
  • Attachment report by file extension for each SMTP host.
  • Attachment report by mime type for each SMTP host.
  • Summary message report for each virtual SMTP instance.
  • Domain report for each virtual SMTP instance.
  • Attachment report by file extension for each virtual SMTP instance.
  • Attachment report by mime type for each virtual SMTP instance.
  • Summary user report for all SMTP addresses.
  • Full user report for all sent messages.
  • Full user report for all received messages.
  • Domain report for each user.
  • Attachment report by file extension for each user.
  • Attachment report by mime type for each user.
More reports will be added to future planned releases and suggestions for reports are always welcome.
