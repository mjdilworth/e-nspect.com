Query - Host

After you have connected to the database server the host will be displayed, together with two containers. One for SMTP Host reports and the other for User Reports.
The available actions from the database host node are as follows:

  • Connect to another database server.
  • Disconnect from the currently selected database server.
  • Create an e-nspect database on the currently selected server (yes you can connect to a database server that does not already have an e-nspect database).
  • Purge data from the e-nspect database on the currently selected server. This is done by selecting a date range.
