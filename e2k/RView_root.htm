Query - Root Node

The "root node" () of e-nspect2000 Query is added to the MMC and is the starting point in your report viewing experience.
The first thing needed to be done is to connect to a database server that hosts the e-nspect database. This is done by selecting the "connect" menu option, as shown below.

This will display the connection dialog box. Enter, or browse for, the database server name and supply the username and password for the connection.
