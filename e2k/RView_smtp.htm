Query - SMTP Host Reports

By selecting and expanding the SMTP Host Reports node you can view many machine specific reports, which are also broken down into the contained virtual SMTP instances.
From the SMTP Host report node you can set the date range for the report generation, print the report, or save the report to a HTML file.
