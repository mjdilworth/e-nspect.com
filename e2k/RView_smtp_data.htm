Query - SMTP Host Reports Data

The reports provide the following information.
Selecting the SMTP Host Reports node will display all SMTP hosts and their summary information. All columns allow for bi-directional sorting which enables quick and easy comparison.

The summary information displayed here consists of:

  • Host Name
  • Msg.Qty - Total number of messages.
  • Msg.Size - Total size (bytes) of all messages.
  • Att.Qty - Total number of attachments.
  • Att.Size - Total size (bytes) of all attachments.
  • Int.Msg.Qty - Total number of local messages.
  • Int.Msg.Size - Total size (bytes) of local messages.
  • Int.Att.Qty - Total number of local attachments.
  • Int.Att.Size - Total size (bytes) of local attachments.
  • Ext.Msg.Qty - Total number of external messages.
  • Ext.Msg.Size - Total size (bytes) of external messages.
  • Ext.Att.Qty - Total number of external attachments.
  • Ext.Att.Size - Total size (bytes) of external attachments.

By selecting the SMTP host node (MAIL1 in the above example) you will get the same information for each virtual SMTP instance on this node.

Each SMTP host and each virtual SMTP instance also has more detailed reports for messages, domains and attachments.
